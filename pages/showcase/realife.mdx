# Realifeサーバー
<br/>
<CustomCallout type="warning">この情報は、一部サーバーと違う可能性があります。<br/>違う箇所を見つけたときは、公式ディスコードのチケットを開き、お知らせください。</CustomCallout>

#### Realifeサーバーは、ロビーにあるNPCをクリックするか、``/server realife`` のコマンドを使用するか、手持ちのゲームセレクターで接続可能です。

---

Realife とは、**Real** + **Life** という、リアルな生活ができることを目標に作り上げられたサーバーです。<br/>
このサーバーには、``McMMO``, ``Towny``, ``Jobs`` のプラグインが入っており、<br/>
我々のチームで開発した、``Realife`` プラグインも入っています。
<br/>
``McMMO`` は、マインクラフトにMMO要素を盛り込むプラグインです。<br/>
``Towny`` は、自分の国や町を作り、たくさんの人といっしょに、自分の国や町を発展させることができるプラグインです。<br/>
``Jobs`` は、自分の職業を選択し、働くことでお金を手に入れることができたりする、お仕事ができるプラグインです。<br/>
<br/>
