# Bridgeサーバー
<br/>
<CustomCallout type="warning">この情報は、一部サーバーと違う可能性があります。<br/>違う箇所を見つけたときは、公式ディスコードのチケットを開き、お知らせください。</CustomCallout>

#### Bridgeサーバーは、ロビーにあるNPCをクリックするか、``/server bridge`` のコマンドを使用するか、手持ちのゲームセレクターで接続可能です。

このBridgeサーバーは、よくPvPをする方や、PvPに詳しい人ならわかる、``橋掛け`` の練習ができるサーバーです。<br/>
橋掛けとは、ブロックで空中をつなぐ行動のことで、SkyWars や BedWars など、<br/>
空にある拠点同士をつなげたり、移動したりすることができます。
<br/>
そんな ``橋掛け`` ですが、最初やるときはとても難しいですし、今では``テリーブリッジ``や、``ゴッドブリッジ`` なども存在しています。<br/>
初心者も、上級者も連取ができるサーバーです。<br/>
橋掛けのタイムスコアは保存され、ランキングとして競い合うことができます。
